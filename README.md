# AI Interview Coach

An AI-powered Interview Preparation Platform built with Java, Spring Boot, PostgreSQL, and AI.

## Tech Stack

- Java 21
- Spring Boot
- PostgreSQL
- Spring Data JPA
- Maven

## Features

- User Authentication (Coming Soon)
- AI Interview Generator
- AI Answer Evaluation
- Resume Analyzer
- Personalized Study Planner
